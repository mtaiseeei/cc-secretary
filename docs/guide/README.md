## 大切な考え方（ひとことで）

- **データは各サービスに置いたまま**参照します。メールや予定の全文を勝手にローカルへコピーしません。
- **記憶と成果物だけ**が、いまいるフォルダの中の**秘書ディレクトリ**（`secretary/`）に残ります。
- 変更は**ローカルに自動でコミット**（作業の区切りを記録）します。インターネットへ送る（push）のは、あなたが頼んだときだけです。
