# Sprint 006 — 公開整備

- Phase: P4（公開整備）
- 主眼: public / MIT リリースに向けたドキュメントとクレジットを整え、第2期カリキュラムへの導線を作る。
- 依存: sprint-001〜005（機能が出揃っていること）

## スコープ（含む）

- 公開ドキュメント（F16）: README（3コマンドインストール・できること・使い方）、使い方 docs。
- クレジット表記: 元作者 Shin-sibainu/cc-company（MIT）を `forkedFrom` で継承・明記。中間フォーク inoshinichi/bootcamp-company は必須クレジットに含めない（constraints 準拠）。
- 第2期カリキュラムへの組み込み導線（座学の語彙・進行との接続を README/docs に反映）。

## スコープ外

- 新機能の追加（本スプリントは整備のみ）。

## 受入基準（薄い契約・詳細化は着手時）

- README から3コマンドで導入でき、非エンジニアが「何ができるか」を日常語で把握できる。
- LICENSE（MIT）とクレジット表記が揃い、元作者への継承が明記されている。
- ドキュメントの文言が非エンジニア体験ルール（日常語＋具体例・3行報告）に沿う。
